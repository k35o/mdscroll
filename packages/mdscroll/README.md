# mdscroll

Push markdown content to a local browser preview, instantly.

```bash
# (not yet published)
npx mdscroll                          # start server + open browser
echo '# hello' | npx mdscroll push    # push content
npx mdscroll push plan.md             # or push a file
```

Designed for AI workflows: pipe a Claude-generated plan into your browser the moment it's ready.
