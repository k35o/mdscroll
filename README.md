# mdscroll

Push markdown content to a local browser preview, instantly.

A lightweight CLI that lets you stream markdown — especially AI-generated plans, drafts, and reports — into a beautifully rendered browser view on localhost. Like unrolling a scroll on your desk.

## Status

Early development. Not yet published to npm.
